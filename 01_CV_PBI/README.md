# CV Interactif (Power BI)

## Présentation
Rapport Power BI interactif conçu pour présenter mon parcours professionnel, mes compétences techniques et mes réalisations clés de manière dynamique.

## Contenu du dossier
* `CV_PBI.pbix` : Rapport source Power BI.
* `CV_PBI.pdf` : Version statique imprimable.
* `demo_cv_pbi.mov` : Démonstration vidéo des fonctionnalités interactives.

## Stack Technique
* **Outil :** Power BI Desktop
* **Langages :** DAX (mesures contextuelles), Power Query (M)
* **Design :** UX/UI orienté lisibilité et navigation fluide
